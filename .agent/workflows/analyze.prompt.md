# Analyze

Perform comprehensive code analysis and provide insights.

## What This Does

- Examines code structure and organization
- Identifies code smells and anti-patterns
- Evaluates adherence to best practices
- Assesses code complexity metrics
- Highlights potential security issues
- Suggests improvement opportunities

## Analysis Areas

### Architecture Analysis

- Component organization and relationships
- Separation of concerns
- Dependency management
- Design pattern usage
- Scalability considerations

### Code Quality Analysis

- Code complexity (cyclomatic complexity)
- Code duplication
- Naming conventions
- Function/method length
- Parameter count
- Nesting depth

### Security Analysis

- Input validation issues
- Authentication/authorization flaws
- Sensitive data exposure
- Injection vulnerabilities
- Dependency vulnerabilities

### Performance Analysis

- Resource usage patterns
- Inefficient algorithms
- Database query optimization
- Caching opportunities
- Memory leaks

## Expected Output

The AI will provide:

- Summary of findings
- Specific issues with code references
- Severity assessment (critical, high, medium, low)
- Actionable recommendations
- Code examples for fixes
