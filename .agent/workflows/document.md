# Document

Generate comprehensive documentation for code and projects.

## What This Does

- Creates documentation for code, APIs, and projects
- Generates README files with proper structure
- Documents functions, classes, and modules
- Explains complex logic and algorithms
- Provides usage examples
- Maintains consistency with project standards

## Documentation Types

### Code Documentation

- Function/method documentation with parameters and return values
- Class documentation with purpose and usage
- Inline comments for complex logic
- API endpoint documentation
- Configuration documentation

### Project Documentation

- README with setup and usage instructions
- Architecture and design decisions
- Contribution guidelines
- Changelog and version history
- Deployment procedures

### API Documentation

- Endpoint descriptions
- Request/response examples
- Authentication requirements
- Error codes and messages
- Rate limits and quotas

## Documentation Standards

- Clear and concise explanations
- Practical examples
- Consistent formatting
- Up-to-date information
- Proper versioning

## Expected Output

The AI will provide:

- Well-structured documentation
- Code examples where appropriate
- Clear explanations of functionality
- Usage instructions
- Best practices and guidelines
